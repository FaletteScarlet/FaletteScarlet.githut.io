Etiquetas de HTML

    Etiquetas de Estructuras Globales

  1. <!DOCTYPE>

    2. Descripción: Esta etiqueta es utilizada para indicar el tipo de documento que se iniciará y la versión de html. Esta línea debe ser la primera línea del archivo, sin espacios o líneas que la precedan. De esta forma, el modo estándar del navegador es activado y las incorporaciones de HTML5 son interpretadas siempre que sea posible, o ignoradas en caso contrario.

    Ejemplo de uso:

    <!DOCTYPE html>

  2. <html>

    3. Descripción: HTML usa un lenguaje de etiquetas para construir páginas web. Esta etiqueta es la raíz del archivo. Estas etiquetas HTML son palabras clave y atributos rodeados de los signos mayor y menor. La mayoría de las etiquetas HTML se utilizan en pares, una etiqueta de apertura y una de cierre, y el contenido se declara entre ellas.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>

    </html>

  3. <head>

    4. Descripción: Esta etiqueta es la cabecera del documento, engloba los metadatos. Dentro de estas etiquetas definiremos el título de nuestra página web, declararemos el set de caracteres correspondiente, proveeremos información general acerca del documento e incorporaremos los archivos externos con estilos, códigos Javascript o incluso imágenes necesarias para generar la página en la pantalla.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>
    <head>

    </head>

    </html>

  4. <body>

    5. Descripción: Esta etiqueta es el cuerpo del documento. Es donde se colocan los elementos visibles de la página.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>
    <head>

    </head>

    <body>

    </body>

    </html>

  5. <meta>

    6. Descripción: Esta etiqueta define el juego de caracteres a utilizar para mostrar el documento. Sirve para especificar como el texto sera presentado en pantalla.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>
    <head>
    <meta charset="iso-8859-1" />

    </head>

    </html>

  6. <title>

    7. Descripción: Esta etiqueta especifica el título del documento. Este titulo sera visible en la ventana del navegador.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>
    <head>
    <meta charset="iso-8859-1" />
    <title> Título </title>

    </head>

    </html>

  7. <link>

    8. Descripción: Esta etiqueta se utiliza para incorporar estilos, códigos Javascript, imágenes o iconos de archivos externos.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>
    <head>
    <meta charset="iso-8859-1" />
    <title> Título </title>
    <link rel="stylesheet" href="misestilos.css">

    </head>

    </html>

  8. <style>

    9. Descripción: Esta etiqueta se utiliza para introducir codigo CSS en line, es decir, en el propio documento HTML.

    Ejemplo de uso:

    <!DOCTYPE html>
    <html>
    <head>
    <meta charset="iso-8859-1" />
    <title> Título </title>
    <style>
    body{
    background-color: black;
    margin: 0 auto;
    }
    </style>

    </head>

    </html>
